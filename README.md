# Wein-Gehirn

Eine cinematische, navigierbare Karte meines Wein-Geschmacks — wo & wann ich welche Weine
getrunken habe, mit einem 3D-Globus der Herkunftsregionen, Geschmacks-Analyse und
begründeten Empfehlungen. Personennamen sind in dieser öffentlichen Version auf Initialen anonymisiert.

Einfach `index.html` öffnen. Datenbasis: persönlicher Vivino-Export.
